2、安装定时插件APScheduler：pip install apscheduler\n
3、安装计划任务需要的插件SQLAlchemy：pip install SQLAlchemy\n
4、安装excel组件xlrd，xlwt\n
5、安装mysql插件mysqlDB：\n
6、安装yaml插件\n
